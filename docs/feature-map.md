- 房间改造入口
- 产品植入入口
- 样板房视频入口
- 响应式 Web/H5 布局
- 上传素材表单
- 空间类型选择
- 装修风格选择
- 运镜风格选择
- 视频比例选择
- 视频时长选择
- 备注要求
- 预计消耗积分
- 确认生成提示
- 我的作品预览
- 今日任务概览
- 租户账户卡片
- 当前会员展示
- 积分账户展示
- 成员席位展示
- 积分流水示例
- 素材库复用能力说明
- 任务记录能力说明
- 作品导出与高清化能力说明
- 异常处理能力说明

### 平台后台

## 上线版需要实现为真实服务的能力

### 多租户与权限

- tenants 表
- users 表
- roles 表
- tenant_id 数据隔离
- 管理员 / 创作者 / 查看者权限判断
- 后台管理员与客户用户权限分离

### 会员与积分

- membership_plans 表
- tenant_memberships 表
- credit_accounts 表
- credit_transactions 表
- 租户共享积分池
- 积分预扣
- 成功确认扣费
- 失败自动退款
- 手动充值与后台调整

### 素材与作品

- materials 表
- works 表
- work_exports 表
- 图片上传
- 视频上传
- 多张参考图上传
- Logo 上传
- 自动缩略图
- 原始版本 / 1080P / 2K / 4K 导出版本

### 任务系统

- generation_tasks 表
- pending_payment
- queued
- generating
- upscaling
- completed
- failed
- refunded
- cancelled
- 自动重试
- 超时处理
- 失败原因记录
- API 返回信息记录

### 提示词系统

- prompt_task_types 表
- prompt_spaces 表
- prompt_styles 表
- prompt_camera_styles 表
- prompt_global_rules 表
- prompt_versions 表
- 平台默认模板
- 租户专属模板
- 提示词版本记录
- 提示词回滚
- 完整提示词快照记录

### 第三方能力

- model_providers 表
- upscale_plugins 表
- 视频生成 API 后端调用
- 高清化插件后端调用
- API Key 后端保存
- 供应商切换
- 失败退款策略

### 系统设置

- system_settings 表
- 平台名称
- 平台 Logo
- 默认水印
- 文件上传限制
- 视频大小限制
- 图片大小限制
- 是否开启注册
- 是否开启邀请成员
